The Lag Ledger

Measure the cold gap between noticing a need and having a tool that still matters when the moment passes.

What it scores

  • Decision-to-tool gap, in minutes or hours.
  • Survival verdict, whether the tool earned a second use.
  • Friction source, setup burden, trust gap, or wrong scope.
  • Sediment, a local log of previous gaps.

Mountain rule

If the tool only works in the demo cave, it is not a tool. It is a weather condition.
